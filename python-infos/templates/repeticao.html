Estruturas de Repetição em Python

Conceito

Permitem executar um bloco de código múltiplas vezes. Principais estruturas:

  • for: Para iterações conhecidas
  • while: Repete enquanto condição for verdadeira

Aplicação

Usadas para:

  • Processar coleções de dados
  • Implementar algoritmos iterativos
  • Repetir ações até condição ser satisfeita

Exemplo de Código


                # For - iterando lista
                frutas = ['maçã', 'banana', 'laranja']
                for fruta in frutas:
                    print(fruta)

                # While - contador
                contador = 0
                while contador < 5:
                    print(contador)
                    contador += 1
