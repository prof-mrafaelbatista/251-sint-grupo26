Vetores e Matrizes em Python

Conceito

Em Python, vetores são implementados como listas e matrizes como listas de listas.

Aplicação

Usados para:

  • Armazenar coleções de dados
  • Operações matemáticas
  • Representação de dados tabulares

Exemplo de Código


            # Vetor (lista)
            numeros = [1, 2, 3, 4, 5]

            # Matriz 3x3
            matriz = [
                [1, 2, 3],
                [4, 5, 6],
                [7, 8, 9]
            ]

            # Acessando elementos
            print(numeros[0])  # Primeiro elemento
            print(matriz[1][2]) # Linha 2, Coluna 3
